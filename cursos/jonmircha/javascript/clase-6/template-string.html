Template String

La concatenación y la interpolación, son tema que se pueden hacer con los strings.

En JavaScript la concatenaciçon se logra con el signo (+), que sirve para unir dos o más valores.

La interpolación, es meter dentro de una cadena de texto, el valor dinamicamente de una variable, se hace con el backtip `${nombre_variable}`.
