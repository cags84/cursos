Propiedad | Display

Con la propiedad display entramos en el concepto de layout model, nos permite definir como se va a comportar una caja con respecto a las que tienen de forma adyacente.

De forma natural exiten elementos en línea y elemento de bloque.

Los valores de esta propiedad son:

  • inline: Establece que la caja se comporta como si fuera un elemento de línea.
  • block: Establece que la caja se comporta como si fueta un elemento en bloque.
  • inline-block: Se comportan como un elemento en línea pero acepta width y height.
  • none: Lo que hace es que oculta el elemento, pero lo sigue renderizando, se tiene que utilizar con cuidado.
  • table: Imita el comportamiento de una tabla.
  • list-item: Imita el comportamiento de una lista.
  • CSS3:
    • flex: Maquetar componentes, por ejemplo un menu, un footer
    • grid: Maquetar layout, acomodar todas las cajas/componentes creados.

Los elementos en línea no aceptan width ni height, aceptan margin y padding, pero solo en el eje horizontal, no en el flujo vertical.
