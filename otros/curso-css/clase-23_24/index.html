Textos y tipografías en CSS

Por normal general el texto es más abundante que las imagenes.

Tipografía

A los tipos de letras que existen se les denomina fuentes o tipografía.

Podemos dividir en dos grupos:

  • Familias tipográficas: Fuentes específicas con nombre (Arial, Times New Romas,etc..
  • Familias genericas: Reciben su nombre en función de sus carácteristicas (serif, san-serif, cursive, monospace, etc....

Cuando elegimos una fuente denemos poner fuentes de reserva-

body { font-family: 'fuente', fallback1, fallback2; }

Las propiedades para cambiar el estilos de la fuentes son varias:

  • font-family
  • font-size
  • font-weight
  • font-style
  • text-transform, todo en mayus o mini...
  • text-align, aliniamiento del texto
  • text-decoration, subrayado, tachado....
  • text-indent, indentación para temas esteticos
  • line-height, establece el alto de línea
  • letter-spacing, establece la separación entre las letras.
