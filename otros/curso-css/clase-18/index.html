Pseudoclases I | Dinámicas, marcadores e idioma

Son selectores que reaccionan en tiempo real detectando la interacción del usuario con algunos elementos, es como colarle una marca a un elemento HTML para saber si el usuario hace algo con ella, y con JavaScript podrimamos realizar acciones sobre esto.

Funcionan con todos los selectores de CSS, los selectores básicos y los compuestos, con todos funciona.

No están atados a ningún elemento.

Su sintaxis es:

Si se le aplica a un selectore en concreto.

selector:pseudoclase { estilos; }

Si se le aplica a todo.

:pseudoclase { estilos; }
