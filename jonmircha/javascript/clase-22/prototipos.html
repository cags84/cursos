Prototipos

JavaScript multiparadigma, (funcional, POO, etc), una caracteristica en JavaScript es que la POO es muy diferentes a los lenguajes como C++ o Java, por ejemplo clases se tiene despues de ES6, sin embargo las clases en JavaScript no es más que una azúcar sintactica, porque detras de camaras, todo son Prototipos.

En POO, hay cuadro conceptos muy importantes:

  • Clases: Modelo a seguir, algo que nos sirve como un esquema para basarnos y generar instancia.
  • Objetos: Es una instancía(copía) de una clase.
    • Atributos: Es una carácteristica o propiedad del objeto, son variables dentro de un objeto.
    • Metodos: Acciones que un objeto puede realizar, son metodos dentro un objeto (son verbos), al fín son funciones dentro de un objeto.
