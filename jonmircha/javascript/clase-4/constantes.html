Constantes

const, es la forma que tenemos en ES6 para declarar constantes.

const nos permite crear variables, que nosotros sepamos que no van a cambiar su valor, durante el flujo de nuestra aplicación.

No se pueden tener constantes VACIAS.

Es un valor constante, no se puede redeclarar o modificar.

A tener en cuenta, con los valores primitivos - no se pueden modificar el valor con el cual se inicializa, pero con los valores compuestos, no puede cambiar el nombre con el que se declaro, pero si el contenido de él, como lo vemos en el ejemplo.

Ejemplo

        const PI = 3.141618;
        const NUEVE = 9;

        console.log(PI);
        console.log(NUEVE);

        // Valores compuestos
        const arreglo = {
            nombre: "Carlos"
        };

        console.log(arreglo.nombre);

        arreglo.nombre = "Otro nombre";
        console.log(arreglo.nombre)
