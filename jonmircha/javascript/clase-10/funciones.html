Funciones

Son bloques de código autocontenido, independiente al scope o ambito global, que se va a definir una sola vez y que se va ejecutar cuantas veces como sea necesaria, puede recibir o no parametros y puede devolver o no resultado.

Las funciones en JavaScript son un objeto, son un tipo de dato muy especial, se les considera "ciudadanos de primera clase", puedes asignarla, puede retornar otra funcion, son muy poderosa desde ES6, como por ejemplo arrow function (funciones de flecha).

Funcion declaradas vs expresadas

El orden en el que se declaran las funciones tienen relevancía dado que, las funciones declaradas pueden invocarse en cualquier parte de nuestro código, incluso antes de que la función sea declarada, lo primero que hace JavaScript es variables y despues funciones.

Las funciones expresada deben ser declaradas antes de mandarlas a llamar.
