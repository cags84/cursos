Position

Flujo de HTML y la propiedad position en CSS

Position es la propiedad que nos permite modifica el flujo de HTML, El flujo en HTML es el orden en el que aparecen los elementos.

los valores de position son:

  • Static (Valor por defecto)
  • Relative
  • Absolute
  • Fixed
  • Sticky

Si creamos el elemento y no le aplicamos algunas de estas opciones (relative, absolute, fixed, sticky), se dice que el elemento no esta posicionado, por defecto es static y se considera que no esta posicionado.

Importante

Cuando un elemento se posiciona podemos moverlo en los 3 ejes:

  • top: Podemos mover el elemento en relación a la parte superior
  • right: Podemos mover el elemento en relación a la parte derecha.
  • bottom: Podemos mover el elemento en relación a la parte inferior.
  • left: Podemos mover el elemento con respecto a la parte izquierda.
  • z-index: Podemos mover el elemento en el eje Z

Nota: Si dos elementos se solapan, osea que un elemento aparece por encima de otro, podemos utilizar z-index, para poder decirle cual debe aparecer primero.
