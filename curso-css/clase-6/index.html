Box Model

En la web todo son cajas, al conjunto de esas cajas y la forma en la que navegador los dibuja se le denomina LAYOUT.


Tips: Todas la etiquetas de HTML se pueden representar con un alto y un ancho, y el navegador lo construye como cajas.


Las propiedades principal de cada una de estas cajas son el ancho (witdth) y el alto (heigth)


Existen dos tipos de elementos en HTML, los elementos en línea (inline) y los elementos en bloque (block), pero estas propiedades se pueden modificar mediante el atributo display


Elementos en línea

  • Son elementos que sólo ocupan el espacio necesario para mostrar su contenido.
  • Si existen varios elementos en línea, uno se pondra al lado del otro.
  • No se puede modificar ni su ancho, ni su alto.

Por ejemplo: a, es un elemento de línea.


Elementos en bloque

  • Ocupan todo el ancho disponible.
  • Se les puede asignar ancho y alto.

Siempre ocupan todo el espacio disponible y a eso se le denomina viewport


En el Modelo de Caja tenemos la siguiente representación:

    _________________________
    |    Margin             | 
    _________________________
    |    Outline            |
    _________________________
    |    Border             |
    _________________________
    |    Padding            |
    _________________________
    |    Contenido          |

Margin: Distancia entre un elemento y otro.
