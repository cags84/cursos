Border CSS

Border, es la línea que bordea la caja, es decir lo que va a continuación del padding, border tambien es un shorthand.

Las propiedades de los border son:

  • Width
  • Style
  • Color

Existen bastantes combinaciones

  • border-width: 2px (top, rigth, bottom, left)
  • border-style: solid dotted; (x, y)
  • border-color: red blue gree; (top X bottom)
  • border-width: 2px 3px 4px 5px; (top rigth bottom left)
