Flexbox - Fundamentos

Es un modelo de layout que permiten que las cajas sean flexible, de hay su nombre.

Existe display:flex y display:inline-flex, para que el contenedor sea bloque o línea respectivamente.

Se necesita un contenedor (contexto) flex-container y al menos un hijo flex-item.

Las cajas flexibles son los items, el contenedor es el contexto al que pertenences los items.

Solo los hijos directos son los flexibles, si tuvireamos otros elementos dentro un un flex-item no aplica.

Un hijo podría ser flex-item y flex-container a la vez si es necesario, para que el elemento dentro del hijo pueda ser un flex-item y el flex-item a su vez flex-container.

Los ::after y los ::before de un elemento con flex, tambien se consideran flex-items.

El texto tambien se considera un hijo del contenedor, se comporta como una caja flexible, lo podemos controlar.

Ejes

Existen dos ejes para colocar y alinear el contenido:

Por defecto el eje principal es horizontal y de izquierda a derecha y el eje secundario es vertical de arriba abajo.

Se pueden intercambiar los ejes y la dirección de ambos.

Estos ejes reciben un nombre:

  • Eje principal: main axis (Eje horizontal)
  • Eje secundario: cross axis (Eje vertical)

A parte se genera un inicio y un fin:

  • Eje principal: main-start, main-end
  • Eje secundario: cross-start, cross-end
