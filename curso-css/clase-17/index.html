Pseudoelementos

Se utilizan para dar estilos a una parte especifica de un elemento.

La sintaxis es: selector::pseudoelemento{ estilos_a_aplicar }

Lo de usar :: (dos puntos doble) es algo que vino en CSS3, para diferenciarlos de las pseudoclases, que se denotan con : (:)

Existen 5 Pseudoelementos:

  • ::first-line (elementos de bloque), sirve para seleccionar la primera línea
  • ::first-letter (elementos de bloque), sirve para seleccionar la primera letra
  • ::selection (retirado de la especicación)
  • Las propiedades ::before, ::after, son particulares:
      • Necesitan la propiedad content es obligatoria y son elemento de línea, por defecto, son hijos del elemento al que pertenecen.
    • ::before
    • ::after

::after, ::before, son elementos que no existen en el HTML y que por consecuencia no existe en el DOM, los pseudoelementos no pertenecen al DOM, ::before se coloca antes del elemento y ::after se coloca despues del elemento.
